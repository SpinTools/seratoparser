# Serato Parser

A GoLang library for reading Serato database files.

Currently supported:
- [x] Database V2
- [x] Crates
- [ ] History File
- [x] History Sessions
- [x] Track Cues
- [x] Track Loops
- [x] Track Color
- [x] Track Flips

## Installation

This package can be installed with the go get command:

```bash
go get -u github.com/SpinTools/serato-parser
```
